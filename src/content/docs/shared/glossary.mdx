# 📚 Glosario de Términos

| Término       | Definición                                                                 |
|---------------|----------------------------------------------------------------------------|
| Flujo         | Secuencia de nodos conectados que automatizan un proceso                  |
| Nodo          | Módulo funcional dentro de un flujo (ej: enviar email, llamar API)         |
| Webhook       | URL que permite iniciar un flujo desde un sistema externo                 |
| Credencial    | Configuración que contiene accesos a servicios externos                   |
| `data`        | Objeto que contiene la información compartida entre los nodos             |
| Trigger       | Evento que inicia la ejecución de un flujo                                |
| Template      | Plantilla de flujo prediseñado que puedes reutilizar                      |
| Execution Log | Registro de pasos ejecutados en un flujo, con resultados                  |
