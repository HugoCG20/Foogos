# 🆘 Support

If you need help with Floogos, you have several options for support.

## 🧑‍💻 Available channels

- Support chat on the platform (if enabled)
- Direct contact via email
- Incident opening from the user panel

## 🛠 Useful information when asking for help.

Please include:
- Flow name
- What you were trying to do
- Error screenshot (if any)
- Approximate time of the event

Our support team is here to help you.
