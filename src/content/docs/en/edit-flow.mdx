## Edit a Flow

You can modify any flow you have already created or cloned.

## How to edit?

1. Go to the **Flows** section.
2. Click on the desired flow
3. The visual editor will open
4. You can move, delete or add modules
5. Edit the configuration of each module from the right panel.
6. Save the changes

## Considerations.
- Only users with permissions can edit
- Changes impact directly if the flow is active
- It is recommended to disable the flow while editing
