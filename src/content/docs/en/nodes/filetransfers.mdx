## Node: File Transfers

This node allows uploading files to a remote server via FTP or SFTP.

## Function
Connects to a server, transfers a file, and completes the operation.

## Configuration

| Field          | Type   | Description                              |
|----------------|--------|------------------------------------------|
| credentials_id | Text   | FTP/SFTP credential ID                   |
| protocol       | Text   | Protocol: ftp or sftp                    |
| localPath      | Text   | Path to the local file                   |
| remotePath     | Text   | Destination path on the remote server    |
