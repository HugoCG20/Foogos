## Node: Schema Parser

This node retrieves the schemas of endpoints from a Swagger JSON file.

## Function
Extracts and structures the components of a REST API from a Swagger definition.

## Configuration

| Field | Type  | Description                           |
|-------|-------|---------------------------------------|
| url   | Text  | URL of the Swagger JSON file          |

## Common Uses
- Automate modeling of external APIs
- Create schemas for quick mapping
