## Node: End

This node indicates the closure of the flow. It requires no additional configuration.

## Function
Terminates the flow execution successfully.

## Configuration

| Field | Type | Description        |
|-------|------|--------------------|
| label | Text | Optional label     |

## Common Usage
- Mark the end of a flow  
- Consolidate multiple branches
