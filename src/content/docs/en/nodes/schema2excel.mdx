## Node: Schema to Excel

This node allows you to convert a data schema into an Excel (.xlsx) file.

## Function
Takes a defined schema and structures it into a file for analysis or documentation.

## Configuration

| Field        | Type   | Description                                 |
|--------------|--------|---------------------------------------------|
| origen       | Text   | Source of the schema                        |
| schema_name  | Text   | Name of the schema to export                |

## Multilanguage
- `help.es`: Genera un fichero excel de schemas  
- `help.en`: Generates file xlsx schema
