## Node: Map Function

This node applies a custom function to each element of a dataset, transforming it.

## Function
Iterates over a dataset and applies a defined function.

## Configuration

| Field         | Type   | Description                                      |
|---------------|--------|--------------------------------------------------|
| functionBody  | Code   | JavaScript code applied to each element          |

## Common Uses
- Transform specific fields
- Generate new values
- Normalize data before saving

## Multilanguage
- `help.es`: Permite mapear un dataset aplicando funciones a los elementos  
- `help.en`: Allows you to map a dataset applying functions to the elements
