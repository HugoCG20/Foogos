## Node: Data Mapper

This node transforms a dataset according to a predefined mapping schema.

## Function
Applies field transformations, renaming, type conversion, and more.

## Configuration

| Field              | Type   | Description                                 |
|--------------------|--------|---------------------------------------------|
| mappingschemas_id  | Text   | ID of the mapping schema to be applied      |

## Common Uses
- Convert supplier data to internal format
- Adapt fields between systems
- Apply automatic structural conversions
