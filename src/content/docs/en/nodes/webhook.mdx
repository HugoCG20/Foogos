## Node: Webhook

This node acts as a trigger to run flows automatically when an external request is received.

## Function
Activates the flow when an HTTP request is made to the URL generated by Floogos.

## Configuration

| Field         | Type  | Description                                    |
|---------------|-------|------------------------------------------------|
| url_webhook   | Auto  | URL automatically generated by the system.     |

## Common Usage
- User signup forms  
- Webhooks from external services (e.g., Stripe, Shopify)  
- Third-party events (CRM, ERP)  

## Recommendations
- Share this URL only with trusted services.  
- View the URL by selecting this node in the flow.
