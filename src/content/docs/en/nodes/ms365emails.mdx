## Node: MS 365 Mails

This node connects to a Microsoft 365 account and retrieves the latest unread emails.

## Function
Queries incoming emails and transforms them into data for the flow.

## Configuration

| Field          | Type   | Description                              |
|----------------|--------|------------------------------------------|
| credentials_id | Text   | Microsoft credential ID                  |
| maxResults     | Number | Number of emails to retrieve             |

## Common Uses
- Automate responses
- Process incoming orders or requests
- Extract information from shared inboxes
