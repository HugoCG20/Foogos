## Node: CSV Create

This node converts a dataset into a CSV file and saves it at a specified location.

## Function
Exports data to CSV from the flow.

## Configuration

| Field     | Type   | Description                         |
|-----------|--------|-------------------------------------|
| filePath  | Text   | Path where the file will be saved   |
| dataKey   | Text   | Key that contains the data          |

## Common Uses
- Save sales reports
- Export processed responses
- Archive customer data
