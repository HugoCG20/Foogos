## Node: ev4Client

This node allows you to connect to your EV4 instance and perform actions on a specific table.

## Function
Acts as a connector to the EV4 system via its internal endpoints.

## Configuration

| Field          | Type   | Description                                      |
|----------------|--------|--------------------------------------------------|
| credentials_id | Text   | ID of the credential to connect to EV4          |
| ev4_table      | Text   | Name of the EV4 table to query or modify        |
| metodotable    | Text   | Specific method to apply on the table           |
| method         | Text   | HTTP method (GET, POST, PUT, etc.)              |
