## Node: WhatsApp - Twilio

This node allows you to send WhatsApp messages using Twilio.

## Function
Sends personalized messages through the verified number in the Twilio account.

## Configuration

| Field          | Type   | Description                                |
|----------------|--------|--------------------------------------------|
| credentials_id | Text   | Twilio credential ID                       |
| phoneNumber    | Text   | Destination phone number                   |
| message        | Text   | Message content to send                    |
