## Node: PDF Extractor

This node extracts text from PDF files, breaking down the content by page if necessary.

## Function
Reads PDF files and converts their content into structured text.

## Configuration

| Field     | Type   | Description                              |
|-----------|--------|------------------------------------------|
| filePath  | Text   | Local path of the PDF file to process    |
| dataKey   | Text   | Key where the result will be stored      |
