## Node: Slack

This node allows you to send a message to a Slack account or channel via Webhook.

## Function
Sends plain or rich-text messages to Slack.

## Configuration

| Field         | Type   | Description                               |
|---------------|--------|-------------------------------------------|
| credentials_id | Text   | Credential ID (optional)                 |
| webhookUrl     | Text   | Webhook URL provided by Slack            |
| message        | Text   | Message content                          |
