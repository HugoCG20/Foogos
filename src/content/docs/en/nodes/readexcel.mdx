## Node: Read Excel

This node allows you to read an `.xlsx` file and convert it into a dataset for the flow.

## Function
Loads structured content from Excel spreadsheets.

## Configuration

| Field     | Type   | Description                                 |
|-----------|--------|---------------------------------------------|
| filePath  | Text   | Path to the Excel file to read              |
| dataKey   | Text   | Storage key for the flow                   |

## Multilanguage
- `help.es`: Lee un fichero xlxs y devuelve el contenido en el data  
- `help.en`: Reads an xlsx file and returns the contents in data
