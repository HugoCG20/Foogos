## Node: Delay

This node allows you to delay the execution of the next step for a specified duration.

## Function
Waits for the configured duration before moving to the next node.

## Configuration

| Field     | Type   | Description                                            |
|-----------|--------|--------------------------------------------------------|
| duration  | Text   | Duration in human-readable format (e.g., 1m, 10s, 2h) |

## Common Usage
- Wait before retrying  
- Pause between API calls  
- Separate process blocks by time
