## Node: HTML to PDF

This node generates a PDF file from HTML content, ideal for reports, invoices, or tickets.

## Function
Sends the HTML to API2PDF for conversion and returns the generated link or file.

## Configuration

| Field     | Type   | Description                                      |
|-----------|--------|--------------------------------------------------|
| apiKey    | Text   | API2PDF key                                      |
| data.html | HTML   | HTML to convert, located in `data.html`          |
