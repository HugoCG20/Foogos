## Node: Send Notification

This node allows you to send push notifications via Firebase Cloud Messaging (FCM).

## Function
Sends a push message with a title and content to the registered device.

## Configuration

| Field          | Type   | Description                                |
|----------------|--------|--------------------------------------------|
| credentials_id | Text   | Firebase credential ID                     |
| title          | Text   | Title of the notification                  |
| message        | Text   | Message or main content                    |

## Common Uses
- System alerts
- Mobile reminders
- Follow-up messages in apps
