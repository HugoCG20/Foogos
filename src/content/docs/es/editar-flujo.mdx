# ✏️ Editar un Flujo

Puedes modificar cualquier flujo que ya hayas creado o clonado.

## 🔄 ¿Cómo editar?

1. Entra a la sección **Flujos**
2. Haz clic sobre el flujo deseado
3. Se abrirá el editor visual
4. Puedes mover, eliminar o agregar módulos
5. Edita la configuración de cada módulo desde el panel derecho
6. Guarda los cambios

## 🔐 Consideraciones
- Solo los usuarios con permisos pueden editar
- Los cambios impactan directamente si el flujo está activo
- Se recomienda desactivar el flujo mientras se edita
