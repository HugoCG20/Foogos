## Crear un Flujo

Los flujos en Floogos se construyen paso a paso utilizando módulos (nodos) conectados entre sí.

## Instrucciones

1. Ir a la sección **Flujos**
2. Hacer clic en **Nuevo flujo**
3. Darle un nombre y descripción
4. Agregar módulos desde el panel izquierdo
5. Conectarlos entre sí arrastrando los conectores
6. Configurar cada módulo haciendo clic sobre él
7. Guardar y Activar

Puedes probar el flujo haciendo clic en **Test** para ver cómo se comporta con datos reales.

## Buenas prácticas
- Comienza con nodos simples como Webhook o Start
- Prueba las decisiones y validaciones con `Decision`
- Usa `End` siempre al final del flujo
