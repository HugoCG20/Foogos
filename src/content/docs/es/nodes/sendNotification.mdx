## Nodo: Send Notification

Este nodo permite enviar notificaciones push a través del servicio Firebase Cloud Messaging (FCM).

## Función
Dispara un mensaje push con título y contenido al dispositivo registrado.

## Configuración

| Campo         | Tipo   | Descripción                             |
|----------------|--------|-----------------------------------------|
| credentials_id | Texto  | ID de credencial de Firebase            |
| title          | Texto  | Título de la notificación               |
| message        | Texto  | Mensaje o cuerpo principal              |

## Uso común
- Alertas de sistema
- Recordatorios móviles
- Mensajes de seguimiento en apps

## Multiidioma
- `help.es`: Envía notificaciones Push mediante FireBase
