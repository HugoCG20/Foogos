## Nodo: Create Excel File

Este nodo guarda un dataset procesado en un archivo Excel en la ruta definida.

## Función
Convierte datos estructurados en hojas de cálculo de Excel.

## Configuración

| Campo     | Tipo   | Descripción                        |
|-----------|--------|------------------------------------|
| filePath  | Texto  | Ruta donde guardar el archivo      |
| dataKey   | Texto  | Clave que contiene el dataset      |

## Uso común
- Generar reportes automáticos
- Guardar resultados de procesos complejos
- Exportar información filtrada

## Multiidioma
- `help.es`: Crea un fichero .xlsx desde un dataset
- `help.en`: Create an .xlsx file based on a dataset
