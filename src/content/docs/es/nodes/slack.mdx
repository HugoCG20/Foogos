## Nodo: Slack

Este nodo permite enviar un mensaje a una cuenta o canal de Slack mediante Webhook.

## Función
Envía mensajes de texto a Slack con formato plano o enriquecido.

## Configuración

| Campo         | Tipo   | Descripción                           |
|----------------|--------|---------------------------------------|
| credentials_id | Texto  | ID de credencial (opcional)           |
| webhookUrl     | Texto  | Webhook URL proporcionado por Slack   |
| message        | Texto  | Contenido del mensaje                 |

## Multiidioma
- `help.es`: Envía mensajes a una cuenta de Slack
- `help.en`: Send messages to a Slack account
