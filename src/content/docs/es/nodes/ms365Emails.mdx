## Nodo: MS 365 Mails

Este nodo se conecta a una cuenta de Microsoft 365 y recupera los últimos correos electrónicos no leídos.

## Función
Consulta correos entrantes y los transforma en datos para el flujo.

## Configuración

| Campo         | Tipo   | Descripción                            |
|----------------|--------|----------------------------------------|
| credentials_id | Texto  | ID de credencial de Microsoft          |
| maxResults     | Número | Cantidad de emails a recuperar         |

## Uso común
- Automatizar respuestas
- Procesar órdenes o solicitudes entrantes
- Extraer información desde bandejas compartidas

## Multiidioma
- `help.es`: Obtiene de ms365 los últimos email sin leer.
- `help.en`: Gets the latest unread emails from MS365.
