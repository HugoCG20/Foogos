## Nodo: ev4Client

Este nodo permite conectarse a tu instancia de EV4 y realizar acciones sobre una tabla específica.

## Función
Actúa como conector con el sistema EV4 a través de sus endpoints internos.

## Configuración

| Campo         | Tipo   | Descripción                                   |
|----------------|--------|-----------------------------------------------|
| credentials_id | Texto  | ID de la credencial para conectarse al EV4   |
| ev4_table      | Texto  | Nombre de la tabla EV4 a consultar o modificar |
| metodotable    | Texto  | Método específico sobre la tabla             |
| method         | Texto  | Método HTTP (GET, POST, PUT, etc.)           |
