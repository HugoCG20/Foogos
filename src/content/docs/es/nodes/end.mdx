## Nodo: End

Este nodo indica el cierre del flujo. No tiene configuración adicional.

## Función
Termina la ejecución del flujo correctamente.

## Configuración

| Campo | Tipo | Descripción        |
|--------|------|--------------------|
| label  | Texto | Etiqueta opcional |

## Uso común
- Marcar fin de un flujo
- Consolidar múltiples caminos

## Multiidioma
- `help.es`: Indica que flujo del workflow ha terminado
- `help.en`: Indicates which workflow has end
