## Nodo: WhatsApp - Twilio

Este nodo permite enviar mensajes de WhatsApp utilizando Twilio.

## Función
Envía mensajes personalizados a través del número verificado en la cuenta de Twilio.

## Configuración

| Campo         | Tipo   | Descripción                           |
|----------------|--------|---------------------------------------|
| credentials_id | Texto  | ID de credencial para Twilio          |
| phoneNumber    | Texto  | Número de destino                     |
| message        | Texto  | Contenido del mensaje a enviar        |

## Multiidioma
- `help.es`: Permite enviar y recibir mensajes a WhatsApp mediante el servicio de Twilio
- `help.en`: Allows you to send and receive WhatsApp messages using the Twilio service.
