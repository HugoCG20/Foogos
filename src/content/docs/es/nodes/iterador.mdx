## Nodo: Iterador

Este nodo permite recorrer un conjunto de datos y ejecutar lógica específica por cada elemento individual.

## Función
Descompone un dataset y procesa cada entrada de forma independiente con módulos conectados.

## Configuración

| Campo           | Tipo   | Descripción                                     |
|------------------|--------|-------------------------------------------------|
| recordModule     | Texto  | Módulo ejecutado por cada iteración            |
| recordConfig     | JSON   | Configuración de ese módulo (url, método, etc) |
| onComplete       | JSON   | Qué hacer al finalizar todas las iteraciones   |

## Multiidioma
- `help.es`: Ejecuta un subflujo por cada registro del dataset
- `help.en`: Executes a subflow for each record in the dataset
