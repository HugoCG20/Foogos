## Nodo: Schema to Excel

Este nodo permite convertir un esquema de datos (schema) en un archivo Excel (.xlsx).

## Función
Toma un esquema definido y lo estructura en un archivo para su análisis o documentación.

## Configuración

| Campo        | Tipo   | Descripción                              |
|--------------|--------|------------------------------------------|
| origen       | Texto  | Fuente del esquema                       |
| schema_name  | Texto  | Nombre del esquema a exportar            |

## Multiidioma
- `help.es`: Genera un fichero excel de schemas
- `help.en`: Generates file xlsx schema
