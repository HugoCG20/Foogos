## Nodo: Schema Parser

Este nodo obtiene los esquemas de endpoints desde un archivo Swagger JSON.

## Función
Extrae y estructura los componentes de un API REST a partir de una definición Swagger.

## Configuración

| Campo | Tipo  | Descripción                         |
|--------|-------|-------------------------------------|
| url    | Texto | URL del archivo JSON de Swagger     |

## Uso común
- Automatizar el modelado de APIs externas
- Crear esquemas para mapeo rápido

## Multiidioma
- `help.es`: Genera de una swagger json obtiene el schema de los end-point
- `help.en`: Generates from a swagger json and obtains the schema of the endpoints
