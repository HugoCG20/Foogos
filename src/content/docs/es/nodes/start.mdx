## Nodo: Start

Inicializa el flujo cuando está programado o lanzado manualmente desde la App.

## Función
Permite comenzar un flujo sin necesidad de trigger externo.

## Configuración

| Campo              | Tipo     | Descripción                                           |
|--------------------|----------|-------------------------------------------------------|
| active_schedule    | Booleano | Si se activa por programación horaria                |
| schedule           | Texto    | Cron string o expresión de tiempo (si aplica)        |
| credentials_id     | Texto    | ID de credencial asociada (opcional)                 |
| data               | JSON     | Objeto de datos inicial (opcional)                   |

## Uso común
- Tareas programadas (ej: cada noche)
- Lanzamientos manuales desde usuarios

## Multiidioma
- `help.es`: Inicializa el flujo
- `help.en`: Initialize the flow
