## Nodo: Evaluate Expression

Este nodo permite ejecutar una expresión o bloque de código JavaScript sobre los datos del flujo.

## Función
Transforma o agrega datos usando lógica personalizada.

## Configuración

| Campo     | Tipo   | Descripción                           |
|-----------|--------|----------------------------------------|
| script    | Código | Bloque de código que accede a `data`   |

## Uso común
- Calcular campos nuevos
- Formatear textos
- Aplicar validaciones personalizadas
