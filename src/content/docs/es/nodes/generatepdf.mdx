## Nodo: HTML to PDF

Este nodo genera un archivo PDF a partir de contenido HTML, ideal para reportes, facturas o tickets.

## Función
Envía el HTML a la API2PDF para convertirlo y devuelve el enlace o archivo generado.

## Configuración

| Campo    | Tipo   | Descripción                                       |
|-----------|--------|---------------------------------------------------|
| apiKey    | Texto  | Clave de API2PDF                                  |
| data.html | HTML   | HTML a convertir, ubicado en `data.html`          |
