## Nodo: Map Function

Este nodo aplica una función personalizada a cada elemento de un dataset, transformándolo.

## Función
Itera sobre un dataset y aplica una función definida.

## Configuración

| Campo         | Tipo   | Descripción                                  |
|---------------|--------|----------------------------------------------|
| functionBody  | Código | Código JavaScript aplicado a cada elemento   |

## Uso común
- Transformar campos específicos
- Generar nuevos valores
- Normalizar datos antes de guardarlos
