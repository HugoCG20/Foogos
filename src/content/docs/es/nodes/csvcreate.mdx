## Nodo: CSV Create

Este nodo convierte un dataset en un archivo CSV y lo guarda en una ubicación especificada.

## Función
Exporta datos a CSV desde el flujo.

## Configuración

| Campo     | Tipo   | Descripción                        |
|-----------|--------|------------------------------------|
| filePath  | Texto  | Ruta donde se guardará el archivo  |
| dataKey   | Texto  | Clave que contiene los datos       |

## Uso común
- Guardar reportes de ventas
- Exportar respuestas procesadas
- Archivar datos de clientes
