## Nodo: Data Mapper

Este nodo transforma un dataset según un mapping schema predefinido.

## Función
Aplica transformaciones de campos, renombrado, conversión de tipos, etc.

## Configuración

| Campo              | Tipo   | Descripción                                 |
|--------------------|--------|---------------------------------------------|
| mappingschemas_id  | Texto  | ID del esquema de mapeo a aplicar           |

## Uso común
- Convertir datos de proveedores al formato interno
- Adaptar campos entre sistemas
- Aplicar conversiones automáticas de estructura

## Multiidioma
- `help.es`: Procesa el mapping de dataset
- `help.en`: Process dataset mapping
