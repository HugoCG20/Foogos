# 🌐 Nodo: Webhook

Este nodo actúa como disparador (trigger) para ejecutar flujos automáticamente cuando se recibe una solicitud desde el exterior.

## 🧩 Función
Activa el flujo cuando se hace una petición HTTP a la URL generada por Floogos.

## ⚙️ Configuración

| Campo         | Tipo   | Descripción                                     |
|---------------|--------|-------------------------------------------------|
| url_webhook   | Auto   | URL generada automáticamente por el sistema.   |

## 📝 Uso común
- Formularios de alta de usuario
- Webhooks de servicios externos (ej. Stripe, Shopify)
- Eventos de terceros (CRM, ERP)

## 💡 Recomendaciones
- Asegúrate de compartir esta URL sólo con servicios de confianza.
- Puedes ver la URL al seleccionar este nodo en el flujo.

## 🌍 Multiidioma
- `help.es`: Activa el trigger de webhook para activar un workflow
- `help.en`: Activate the webhook trigger to activate a workflow
