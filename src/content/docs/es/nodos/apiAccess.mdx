# 🌐 Nodo: API Access

Este nodo permite conectarse a un servicio web externo (API) y obtener datos para ser usados dentro del flujo.

## 🧩 Función
Hace una petición HTTP (GET, POST, etc.) a una URL y guarda la respuesta.

## ⚙️ Configuración

| Campo           | Tipo    | Descripción                                   |
|------------------|---------|-----------------------------------------------|
| url              | Texto   | URL del servicio API                          |
| method           | Select  | Verbo HTTP: GET, POST, PUT, DELETE...         |
| credentials_id   | Texto   | ID de credencial para autorización (opcional) |

## 📝 Uso común
- Consultar catálogos de productos
- Obtener datos de clientes desde servicios externos
- Validar tokens o respuestas en tiempo real

## 🌍 Multiidioma
- `help.es`: Accede a un end-point y obtiene la información asignándola al contenido a un data
- `help.en`: Access an endpoint and obtain information by assigning content to a data
