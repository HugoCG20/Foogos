# 🆘 Soporte

Si necesitas ayuda con Floogos, tienes varias opciones para obtener soporte.

## 🧑‍💻 Canales disponibles

- Chat de soporte en la plataforma (si está habilitado)
- Contacto directo vía email
- Apertura de incidencia desde el panel de usuario

## 🛠 Información útil al pedir ayuda

Por favor incluye:
- Nombre del flujo
- Qué estabas intentando hacer
- Captura de error (si hay)
- Hora aproximada del evento

Nuestro equipo de soporte está para ayudarte.
